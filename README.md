## Installation

Not needed.

## Usage

```bash
$ $ npm init @adobe/franklin-service
```

`create-franklin-service` will ask you a couple of questions and create a ready-to-go directory

## Development

### Build

```bash
$ npm install
```

### Test

```bash
$ npm test
```

### Lint

```bash
$ npm run lint
```
